# Procesador de Datos
Proyecto usando las tecnologías de python para procesar datos y filtrar los mas criticos generando un reporte en .txt, .json

# Tecnologias
|Tecnologías|Versión|Proposíto|
|-|-|-|
|Python|3.10.12|Ejecutar el script y la logica|
# Requisitos
- Python 3.10.12 o superior
# Instalacion y uso
- Clonar repositorio
``` bash
git clone https://github.com/slim-carrasco/archivos-script.git
```
- Para correr el script
``` python
python3 procesar.py
```

# Limitaciones
-  El script solo funciona con CSVs que tengan exactamente estas columnas: timestamp, servidor, cpu, memoria, status
